Pertanyaan umum

Jawaban singkat tentang format, proses, hasil, dan privasi.

Format apa yang didukung?+

JPG/JPEG, PNG, dan WebP statis. SVG, GIF animasi, TIFF, PDF, arsip, dan file rusak ditolak.

Berapa lama pemrosesan berlangsung?+

Tergantung resolusi gambar, mode, kemampuan CPU, dan panjang antrean. Aplikasi tidak menampilkan persentase palsu atau menjanjikan waktu tertentu.

Apa beda mode Cepat dan Kualitas?+

Cepat memakai model ringan u2netp yang cocok untuk VPS kecil. Kualitas memakai isnet-general-use bila tersedia dan biasanya membutuhkan lebih banyak waktu serta memori.

Apakah gambar saya bersifat publik?+

Tidak. Akses file membutuhkan kepemilikan sesi browser atau hak staff. Object storage harus dikonfigurasi private.

Kapan file dihapus?+

Default setelah {{ config.retention_hours }} jam, atau lebih cepat bila Anda menekan Hapus. Metadata job dapat disimpan lebih lama.

Mengapa upload ditolak?+

File mungkin terlalu besar, format dan isi tidak cocok, animasi, rusak, resolusi di luar batas, rate limit tercapai, worker offline, atau antrean penuh.
