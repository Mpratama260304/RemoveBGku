KEBIJAKAN PRIVASI

Privasi yang dijelaskan apa adanya.

Terakhir diperbarui: 22 Juli 2026. Dokumen ini menjelaskan perilaku instalasi default; operator dapat menyesuaikan infrastruktur.

Pemrosesan server-side

Gambar yang Anda pilih dikirim ke server dan diproses oleh model AI lokal pada server. Pemrosesan tidak berlangsung hanya di perangkat Anda.

Retensi dan penghapusan

File normalnya disimpan selama {{ config.retention_hours }} jam lalu dihapus otomatis. Anda dapat menghapus file lebih cepat melalui halaman Riwayat. Job yang dipin administrator untuk kebutuhan operasional sah dapat disimpan lebih lama, sementara metadata teknis dapat dipertahankan setelah file dihapus.

Data teknis terbatas

Layanan menyimpan nama file yang disanitasi, ukuran, dimensi, status proses, hash file, hash HMAC sesi dan alamat IP, serta user-agent terbatas. Raw IP, isi cookie, password, dan secret tidak disimpan bersama job.

Penyimpanan pihak ketiga

Operator dapat mengonfigurasi object storage pihak ketiga yang kompatibel S3. Bucket harus private. Aplikasi ini tidak menjual gambar pengguna.

Batas jaminan

Tidak ada sistem yang dapat menjanjikan keamanan atau kerahasiaan absolut. Jangan unggah dokumen yang sangat sensitif. Untuk penggunaan komersial, operator sebaiknya meminta tinjauan hukum independen.
